# Math Dash Mania（数学冲刺狂）

一个专注于心算训练的移动端优先小游戏。通过精美的UI设计、简单易懂的玩法、科学的难度梯度设计，让用户在娱乐中提升数学心算能力。

其实还可以用简单小游戏的方式，集中注意力降低焦虑。

## 🚀 核心特性

- **移动端优先设计** - 响应式界面，完美适配手机和平板
- **精美的UI界面** - 现代简约设计，支持深色/浅色主题切换
- **智能难度调节** - 多种难度等级（简单/中等/困难/专家）
- **丰富的游戏模式** - 加减乘除四则运算，多种时长选择
- **完整成就系统** - 15+ 种成就，包含普通、稀有、史诗、传奇四个稀有度
- **个性化定制** - 头像、主题、称号系统，可解锁多种个性化内容
- **音效系统** - 完整的音效和背景音乐支持，可自定义音量设置
- **详细数据统计** - 学习进度分析、运算符专项统计、历史记录追踪
- **PWA支持** - 支持离线使用和安装到桌面
- **性能优化** - 懒加载、代码分割、缓存策略

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式系统**: Tailwind CSS
- **动画库**: Framer Motion
- **状态管理**: Zustand（持久化存储）
- **路由**: React Router DOM
- **图标**: Lucide React
- **音频**: Howler.js
- **图表**: Recharts
- **PWA**: Vite PWA Plugin + Workbox

## 📦 安装和运行

1. 克隆项目
```bash
git clone <repository-url>
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

5. 预览生产版本
```bash
npm run preview
```

6. 部署到 GitHub Pages
```bash

```

## 🎮 游戏玩法

### 基础玩法
1. **开始游戏** - 在主页选择游戏模式和难度
2. **答题模式** - 支持输入模式和选择题模式
3. **计时挑战** - 在限定时间内尽可能多地答题
4. **连击系统** - 连续答对可获得额外积分
5. **生命系统** - 答错会减少生命值，生命耗尽游戏结束

### 游戏设置
- **时长选择** - 30秒快速模式或自定义时长
- **难度等级** - 简单、中等、困难、专家四个等级
- **运算类型** - 加法、减法、乘法、除法可单独或组合选择

## 🏆 成就系统

### 成就类型
- **基础成就** - 初来乍到、百分达人
- **技能成就** - 速算高手、数学大师、连击王
- **专精成就** - 加法专家、乘法大师、精准射手
- **挑战成就** - 完美主义者、马拉松跑者、极速竞赛者
- **传奇成就** - 连胜大师、世纪得分者、百战老兵

### 奖励系统
- **积分奖励** - 解锁成就获得积分
- **头像奖励** - 解锁特殊头像
- **主题奖励** - 解锁独特主题
- **称号奖励** - 获得专属称号

## 🎨 个性化系统

### 头像系统
- 多种风格头像可选
- 通过成就解锁稀有头像
- 支持普通、稀有、史诗、传奇四个稀有度

### 主题系统
- 深色/浅色模式切换
- 多种颜色主题
- 成就解锁特殊主题

### 称号系统
- 基于游戏表现解锁称号
- 显示个人游戏风格
- 可在个人资料中展示

## 🎵 音效系统

### 音效类型
- **正确答题音效** - 清脆的成功提示音
- **错误答题音效** - 温和的错误提示
- **界面交互音效** - 按钮点击音效
- **成就解锁音效** - 特殊成就提示音
- **游戏开始/结束音效** - 游戏状态音效

### 音频设置
- 独立的音效/音乐音量控制
- 支持完全静音模式
- 使用 Mixkit 免费音效资源

## 📊 数据统计

### 游戏统计
- **总体数据** - 总游戏局数、总得分、最高分
- **准确率统计** - 整体正确率、平均用时
- **连击记录** - 最佳连击、最长连击
- **时间统计** - 总游戏时间、最快完成时间

### 专项统计
- **运算符统计** - 加减乘除各自的正确率
- **每日坚持** - 连续游戏天数追踪
- **历史记录** - 最近50局游戏详细记录
- **进步追踪** - 个人能力发展曲线

## 📱 移动端优化

- **响应式设计** - 适配各种屏幕尺寸
- **触摸优化** - 友好的触摸交互体验
